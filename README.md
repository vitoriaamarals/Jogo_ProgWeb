# 🐍 Jogo da Cobrinha

## Projeto de Desenvolvimento de Jogo Web - Programação Web (GAC116) - UFLA - 2026/1

### O jogo pode ser acessado por aqui: [🔗](https://vitoriaamarals.github.io/Jogo_ProgWeb/)

## 🎮 Como Jogar

O objetivo do jogo é acumular o máximo de pontos possível.<br/>
- **Setas do Teclado:** Controlam a direção da serpente.
- **Barra de Espaço:** Pausa e despausa o jogo.
- **Pontuação:** Cada maçã coletada vale 10 pontos.
- **Nível:** A cada 50 pontos, o nível aumenta e a velocidade da serpente sobe.
- **Derrota:** Ocorre se a serpente colidir com as bordas da grade ou com o próprio corpo. <br/>

## 🛠️ Requisitos Técnicos

- **Tecnologias:** Desenvolvido exclusivamente com HTML, CSS e JavaScript.
- **Lógica de Jogo:** Implementação de sistemas de pontuação, progressão de níveis, estados de pausa e reinício.
- **Manipulação do DOM:** Criação dinâmica da grade de jogo (20x20) e controle de elementos de interface via JavaScript.

## 📂 Organização do Projeto

O repositório está organizado da seguinte forma:

- `index.html`: Estrutura principal da página e elementos de interface.
- `styles.css`: Estilização visual, animações neon e layout responsivo.
- `script.js`: Toda a lógica do motor do jogo, eventos e estados.
